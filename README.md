# drone-autonomy

Learning autonomous drone development using ArduPilot SITL simulation, with the goal of building Python-based autonomous missions and eventually deploying to physical hardware.

## Setup
- Mission Planner (Windows) for ground station and SITL launcher
- ArduPilot SITL for simulated multirotor
- Python with pymavlink/DroneKit for mission scripting (coming soon)

## Goals
- Phase 1: Scripted autonomous missions in simulation (takeoff, waypoint navigation, landing)
- Phase 2: Computer vision integration with OpenCV for target detection
- Phase 3: Deploy to physical quadcopter hardware
